# OpenRouter Free Model Notes

Working note captured on 2026-07-15 from user research. This document is a shortlist for experimentation, not a verified source of truth. Before relying on any model, confirm that the OpenRouter model id still exists and still ends in `:free`.

## Shortlist by Use Case

- General reasoning and tool use: `GPT-OSS 120B`
- Faster lightweight general reasoning: `GPT-OSS 20B`
- Long-horizon orchestration and research: `Nemotron-3 Ultra`
- Long-context efficient alternative: `Nemotron-3 Super`
- Stable multilingual drafting: `Llama 3.3 70B`
- Stable non-thinking general output: `Qwen3 Next 80B`

## Notes

- The research note highlights `Qwen3 Coder` and `GPT-OSS 120B` as the strongest current free options for coding and general reasoning.
- Several shortlisted models reportedly offer very large context windows, including up to 1M tokens.
- Free-tier limits may be around 20 requests per minute and 200 requests per day, so batch generation is safer than one LLM request per resume.
- Availability, quotas, and pricing can change without notice, so this file should be treated as planning guidance only.
